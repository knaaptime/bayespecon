    bayespecon.logdet.LogDetMethod¶

    class bayespecon.logdet.LogDetMethod(*values)[source]¶

    Canonical names for the log-determinant approximation methods.

    Each member is a string so the enum members can be used interchangeably with their string values (e.g. LogDetMethod.EIGENVALUE == "eigenvalue").

If there is a third argument, it must be a string, whose characters will be mapped to None in the result.
